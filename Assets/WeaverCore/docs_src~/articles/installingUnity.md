# Downloading and Installing Unity

## Step 1 : Download and Install Unity Hub

Download the Unity Hub Here (https://unity3d.com/get-unity/download)

![Hub Website](https://user-images.githubusercontent.com/12601671/141714396-1c4b82e5-6c90-489c-a31e-e852b118f43f.PNG)

![Hub Part 1](https://user-images.githubusercontent.com/12601671/141714712-03a25dd3-5a63-46b3-9d23-1c1993cdb515.PNG)

![Hub Part 2](https://user-images.githubusercontent.com/12601671/141714707-aa8f983d-3556-46e7-a9cf-0fc05cd13fca.PNG)

![Hub Part 3](https://user-images.githubusercontent.com/12601671/141714710-625decf7-366b-43d1-aca2-5d8cb2199baa.PNG)

### When finished installing, launch the program and you should see something like this:

### **NOTE: If it prompts you to upgrade to the new 3.0 Hub, don't upgrade**

![Hub Installed](https://user-images.githubusercontent.com/12601671/141717782-b2a33a2d-7587-40b6-9605-27b53c9c4ff8.PNG)

### From here, click on "Manage License" at the bottom-right corner:

![Hub Manage License](https://user-images.githubusercontent.com/12601671/141717842-a0aa8064-a59b-4a6e-b1f9-c1bf61b18259.PNG)

### If you aren't logged in yet, it will prompt you to login. If you see this screen, click on the "login" button, and you will be brought to the login screen:

![Hub Login](https://user-images.githubusercontent.com/12601671/141717915-f20ff9b6-68fe-4bec-9287-5eab79c71971.PNG)

### From Here, you can either login to an existing account, or create an account to login to. You can also login using your Google or Facebook account using the buttons at the bottom (which should make the process much easier)

![Hub Login Screen](https://user-images.githubusercontent.com/12601671/141718008-434b6b79-4cdf-41b3-a241-1b6b1b4c1e04.PNG)


### Once logged in, you can then click on "Activate License"

![Hub Logged In](https://user-images.githubusercontent.com/12601671/141718640-df2e1b1b-2104-4098-bf61-0b9069ba7dcf.PNG)

### For the License, pick "Unity Personal" and then click on "I don't use Unity in a professional capacity". Then click "Done"

![Hub License Registration](https://user-images.githubusercontent.com/12601671/141718700-88bd57e9-f1ef-47b8-83ee-ef5f1821e73c.PNG)

### When that's done, you can exit out of the "Preferences" menu, and the Hub is successfully installed

![Hub Exit Preference](https://user-images.githubusercontent.com/12601671/141718787-a71b11a9-809e-4d9d-a2b7-5e4e0c690fc2.PNG)

## Step 2 : Installing Unity Editor 2020.2.2f1

### [Click Here](https://download.unity3d.com/download_unity/068178b99f32/Windows64EditorInstaller/UnitySetup64-2020.2.2f1.exe) to download the Unity 2020.2.2f1.

### You can also visit the [Unity Download Archive](https://unity3d.com/get-unity/download/archive) and find it there too

### When downloaded, start the installation process

![Editor Install 1](https://user-images.githubusercontent.com/12601671/141719476-751eed15-72b6-44aa-bbe4-df167a550157.PNG)

![Editor Install 2](https://user-images.githubusercontent.com/12601671/141719481-7b56abf8-7149-48ab-8b0b-7c52028835e6.PNG)

![Editor Install 3](https://user-images.githubusercontent.com/12601671/141719479-7bff938a-1dfe-4715-8ab8-2e416c64ccca.PNG)

## Remember where you installed the Editor. We will need it later

![Editor Install 4](https://user-images.githubusercontent.com/12601671/141719478-ba663bbc-53cb-44df-81ad-c51c68f00d13.PNG)


![Editor Install 5](https://user-images.githubusercontent.com/12601671/141719639-e7e34101-e896-4f98-a942-19c94aec8f0b.PNG)

### The install may take a while

![Editor Install 6](https://user-images.githubusercontent.com/12601671/141719655-6ab7adea-8731-4a93-b30e-f05ef7ee65a8.PNG)


### When the install is finished, close the window

![Editor Install 7](https://user-images.githubusercontent.com/12601671/141720043-bac63be9-7c8f-48bd-a763-c5f03dcc84ec.PNG)


### Now, open up the Unity Hub, and go to the "Installs" section

![Hub Installs Section](https://user-images.githubusercontent.com/12601671/141720209-35f8e086-bad7-4b33-94ad-32f6068e1494.PNG)

### Now, to add the Unity Editor we just installed, go to the "Locate" Button

![Hub Locate](https://user-images.githubusercontent.com/12601671/141720350-78f24984-9dd1-4a05-89e7-0694cd8046cb.PNG)

### Remember where you installed your Unity Editor from earlier? Go to the place where your Unity Editor 2020.2.2 is installed, select the "Unity.exe" file, and click on "Select Editor"

![Hub Select Editor Location](https://user-images.githubusercontent.com/12601671/141720542-bfbadaa6-603b-47fa-81c1-020cc817fc63.PNG)

### If you see this, then your Editor has been successfully added to the Hub and your installation is complete!

![Hub Editor Added to Hub](https://user-images.githubusercontent.com/12601671/141720656-2fe0f06f-b960-494b-a098-1dc67648e8fe.PNG)
